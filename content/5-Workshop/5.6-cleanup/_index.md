To avoid incurring costs, clean up resources after finishing the workshop:

#### 1. Terminate EC2
+ Go to **EC2 Dashboard** > **Instances**.
+ Select instance `workshop-aws-streamlit`.
+ Choose **Instance State** > **Terminate instance**.
   
![Terminate EC2](/images/5-Workshop/5.6-Cleanup/stop.png)

#### 2. Delete Security Group
(Optional) Delete the Security Group if no longer needed.
